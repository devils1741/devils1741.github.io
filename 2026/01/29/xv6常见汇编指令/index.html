risc-v常见汇编指令

发表于2026-01-29|更新于2026-01-31
|浏览量:
  1. addi
addi rd, rs1, imm

加法操作，rd = rs1+imm
rd：目标寄存器，存放结果
rs1：源寄存器，参与加法运算
imm：立即数（12 位有符号整数，-2048 ~ 2047）说人话就是指令里直接写死的常数

addi sp,sp,-48   // 栈顶向下移动48个字节
  1. sd
sd rs2, offset(rs1)

把寄存器rs2的数据存储到内存中

sd：store double（存 8 字节整数到内存）
rs2：源寄存器，需要写入内存的值
rs1：基址寄存器，存储的目标地址基准，说人话就是栈顶
offset = 偏移量（立即数，单位字节），偏移量需要字节对齐不然系统会报错

sd ra,40(sp) // 把寄存器ra的值存储到sp中。40代表偏移量。

3. mv
```asm
mv rd, rs

把数据从rs复制搭配rd，只能操作寄存器
注意这是个伪指令，实际上应该是

addi rd, rs, 0

mv = move（寄存器拷贝，伪指令）
rd = 目标寄存器
rs = 源寄存器

  1. auipc
    auipc rd, imm
    把 PC（当前指令地址）加上一个立即数左移12位，然后存到寄存器rd
auipc s2, 0x1 // 等价于s2 = PC + 0x1000
  1. li
li rd, imm

把立即数imm的值直接加载到寄存器 rd
也是伪指令，等价于

// 立即数小：-2048 ~ 2047
addi t0, x0, 10

或者

// 立即数大：超过 12 位
lui t0, 0x12345       # 高 20 位
addi t0, t0, 0x678    # 低 12 位补上
  1. bge
bge rs1, rs2, label

bge(branch if greater or equal)
如果 rs1 ≥ rs2则跳转到label

bge	a5,a0,c0 <main+0x4e>
  1. addiw
addiw rd, rs1, imm

把32位的寄存器值与12位立即相加，然后符号拓展到64位寄存器rd
C语言实现：rd = (int64_t)((int32_t)rs1 + imm);

addiw	s3,a0,-2。 // 32位寄存器a0的值-2之后存储到64位寄存器s3中
  1. slli
slli rd, rs1, shamt

把 rs1 的值逻辑左移 shamt 位，然后存入 rd。寄存器值左移固定位数，低位补0。常用于乘以2的幂或计算偏移地址
C语言实现：rd = rs1 << shamt;

  1. jal
jal rd, label

jal = Jump And Link
跳转到label地址,把下一条指令地址（PC+4）保存到 rd
等价C语言实现

rd = PC + 4;   // 保存返回地址
PC = label;    // 跳转
  1. ld
ld rd, offset(rs1)

Load Doubleword, 从内存读取8个字节（64位）到寄存器rd。内存地址 = rs1 + offset
